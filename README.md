# UCM-badges

A quick way to find which badges a course satisfies.

## Examples
```console
foo@bar:~$ ./badges.out 
Enter Course: math-32

MATH 032

Badges:
Scientific Method
Quantitative and Numerical Analysis
Practical and Applied Knowledge

General Ed:
```
